Probability distributions¶

A set of utilities for cmputing probability and cumulative density functions

equadratures.distributions.CDF_BetaDistribution(N, a, b, lower, upper)¶

A beta cumulative density function.

Parameters:
  • N (integer) – Number of points for defining the cumulative density function.
  • a (double) – First shape parameter of the beta distribution. This value has to be greater than 0.
  • b (double) – Second shape parameter of the beta distribution. This value has to be greater than 0.
  • lower (double) – Lower bound of the support of the beta distribution.
  • upper (double) – Upper bound of the support of the beta distribution.
Returns:

An array of N equidistant values over the support of the distribution.
Returns:

Cumulative density values along the support of the beta distribution.

equadratures.distributions.PDF_CustomDistribution(N, data)¶

A probability density function associated with a given data set.

Parameters:
  • N (integer) – Number of points for defining the probability density function.
  • data (array) – An array of data samples from which a custom distribution is generated.
Returns:

An array of N equidistant values over the support of the distribution.
Returns:

Probability density values along the support of the custom distribution.

Notes:

To obtain a probability density function from finite samples, this function uses kernel density estimation (with a Gaussian kernel).
