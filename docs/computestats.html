Statistics¶

Computing Statistics from Polynomial Expansions

class effective_quadratures.computestats.Statistics(coefficients, index_set)¶

This class defines a Statistics object

Parameters:
  • matrix coefficients (numpy) – polynomial coefficients (can be from any of the methods)
  • index_set (IndexSet) – The index set corresponding to the polynomial basis that was used to compute the coefficients
getFirstOrderSobol()¶

Computes the first order Sobol indices.

Parameters:self (Statistics) – An instance of the Statistics class
Returns:first_order_sobol_indices
Return type:numpy ndarray
getMean()¶

Computes the mean of a polynomial expansion using its coefficients.

Parameters:self (Statistics) – An instance of the Statistics class
Returns:mean
Return type:double

Notes The mean is simply the first coefficient of the expansion.

getVariance()¶

Computes the variance of a polynomial expansion using its coefficients.

Parameters:self (Statistics) – An instance of the Statistics class
Returns:variance
Return type:double

Notes The variance is the sum of the squares of all the coefficients except the first coefficient.
