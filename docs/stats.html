Statistics utilities¶

Computing Statistics from Polynomial Expansions

class equadratures.stats.Statistics(coefficients, basis, parameters, quadrature_points=None, quadrature_weights=None, polynomial_evals=None)¶
Parameters:
  • coefficients (numpy-matrix) – Coefficients from a polynomial expansion. Can be computed using any technique.
  • basis (Basis) – Polynomial index set. If an index set is not given, the constructor uses a tensor grid basis of polynomials. For total order and hyperbolic index sets, the user needs to explicity input an index set.
Attributes:
  • self.mean: (double) Mean of the polynomial expansion.
  • self.variance: (double) Variance of the polynomial expansion.
  • self.sobol:(dict) Sobol’ indices of order up to number of dimensions.

Notes: In a future release we will be incorporating second order Sobol indices, skewness and kurtosis based indices. Stay tuned!

getCondKurtosis(order=1)¶

Get conditional kurtosis indices at specified order.

Parameters:int (order) – The order at which conditional kurtosis indices are computed. By default, computes first order conditional kurtosis.
Returns:indices, Dictionary where keys specify non-zero dimensions and values represent conditional kurtosis indices.
Return type:dict

Sample usage: stats = Statistics(coeffcients, basis) first_order_kurtosis = stats.getCondKurtosis(1)

plot(filename=None)¶

Produces a bar graph of the first order Sobol indices

Parameters:
  • object (Statistics) – An instance of the Statistics class.
  • filename (string) – A file name in case the user wishes to save the bar graph. The default output is an eps file.
