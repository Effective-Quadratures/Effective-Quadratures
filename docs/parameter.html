Parameter¶

Core class for setting the properties of a univariate parameter.

class equadratures.parameter.Parameter(order, lower=None, upper=None, param_type=None, shape_parameter_A=None, shape_parameter_B=None, data=None)¶

This class defines a univariate parameter. Below are details of its constructor.

Parameters:
  • lower (double) – Lower bound for the parameter.
  • upper (double) – Upper bound for the parameter.
  • order (integer) – Order of the parameter.
  • param_type (string) – The type of distribution that characterizes the parameter. Options include: Gaussian, Truncated-Gaussian, Beta, Cauchy, Exponential, Uniform, Gamma, Weibull. If no string is provided, a Uniform distribution is assumed. If the user provides data, and would like to generate orthogonal polynomials (and quadrature rules) based on the data, they can set this option to be Custom.
  • shape_parameter_A (double) – Most of the aforementioned distributions are characterized by two shape parameters. For instance, in the case of a Gaussian (or TruncatedGaussian), this represents the mean. In the case of a Beta distribution this represents the alpha value. For a uniform distribution this input is not required.
  • shape_parameter_B (double) – This is the second shape parameter that characterizes the distribution selected. In the case of a Gaussian or TruncatedGaussian, this is the variance.
  • data – A numpy array with data values (x-y column format). Note this option is only invoked if the user uses the Custom param_type.

getJacobiEigenvectors(order=None)¶

Returns the eigenvectors of the tridiagonal Jacobi matrix. These are used for computing quadrature rules for numerical integration.

Parameters:
  • self (Parameter) – An instance of the Parameter class.
  • order (int) – Number of eigenvectors required. This function makes the call getJacobiMatrix(order) and then computes the corresponding eigenvectors.
Returns:

A order-by-order matrix that contains the eigenvectors of the Jacobi matrix.

An order-by-order sized Jacobi tridiagonal matrix.
