IndexSet¶

Index sets for multivariate polynomials

class effective_quadratures.indexset.IndexSet(index_set_type, orders, level=None, growth_rule=None, dimension=None)¶
This class defines an index set
Parameters:
  • index_set_type (string) – The type of index set to be used. Options include: Total order, Tensor grid, Sparse grid and Hyperbolic basis.
  • of integers orders (ndarray) – The highest polynomial order in each direction
  • level (integer) – For sparse grids the level of the sparse grid rule
  • growth_rule (string) –
    The type of growth rule associated with the sparse grid. Options include:
    linear and exponential.

    Sample declarations

    >> IndexSet('Tensor grid', [3,3,3])
>> IndexSet('Sparse grid', )
getCardinality()¶

Returns the cardinality (total number of elements) of the index set

Parameters:
  • self (Parameter) – An instance of the Parameter class
  • order (int) – Number of eigenvectors required. This function makes the call getJacobiMatrix(order) and then computes the corresponding eigenvectors.
Returns:

V, order-by-order matrix that contains the eigenvectors of the Jacobi matrix
Return type:

ndarray

Sample declaration

# Code to Jacobi eigenvectors
>> var4 = Parameter(points=5, param_type='Gaussian', shape_parameter_A=0, shape_parameter_B=2)
>> V = var4.getJacobiEigenvectors()
