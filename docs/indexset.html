IndexSet¶

Index sets for multivariate polynomials

class effective_quadratures.indexset.IndexSet(index_set_type, orders=None, level=None, growth_rule=None, dimension=None, q=None)¶
This class defines an index set
Parameters:
  • index_set_type (string) – The type of index set to be used. Options include: Total order, Tensor grid, Sparse grid and Hyperbolic basis.
  • of integers orders (ndarray) – The highest polynomial order in each direction
  • level (integer) – For sparse grids the level of the sparse grid rule
  • growth_rule (string) –
    The type of growth rule associated with the sparse grid. Options include:
    linear and exponential.

    Sample declarations

    >> IndexSet('Tensor grid', [3,3,3])
>> IndexSet('Sparse grid', level=3, growth_rule=5, dimension=3)
>> IndexSet('Total order' [3, 3, 3])
>> IndexSet('Hyperbolic basis', [3,3], q=0.75)
