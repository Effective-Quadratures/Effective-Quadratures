Dimension Reduction Utilities¶

Dimension Reduction Functionalities

equadratures.dimension_reduction.computeActiveSubspaces(PolynomialObject, samples=None)¶

Computes the eigenvalues and corresponding eigenvectors for the high dimensional kernel matrix

Parameters:
  • PolynomialObject – an instance of PolynomialObject class
  • samples – ndarray, sample vector points, default to None
Returns:

  • eigs (numpy array): Eigenvalues obtained
  • eigVecs (numpy array): Eigenvectors obtained
equadratures.dimension_reduction.jacobian(V, V_plus, U, y, f, Polybasis, eta, minmax, X)¶

Internal function to variable_projection Calculates the Jacobian tensor using polynomial basis functions

Parameters:
  • V – ndarray, the affine transfored outputs
  • V_plus – ndarray, psuedoinverse matrix
  • U – ndarray, the active subspace directions
  • y – array, the untransformed projected values of inputs in active subspace
  • f – array, the untransfored outputs
  • Polybasis – Poly object, an instance of Poly class
  • eta – ndarray, the affine transformed projected values of inputs in active subspace
  • minmax – ndarray, the upper and lower bounds of input projections in each dimension
  • X – ndarray, the input
Returns:

  • J (ndarray): The Jacobian tensor
equadratures.dimension_reduction.linearModel(Xtrain, ytrain, bounds)¶

Computes the coefficients for a linear model between inputs and outputs

Parameters:
  • Xtrain – ndarray, the input values
  • ytrain – array, the output values
  • bounds – ndarray, the upper and lower bounds for input values in each dimension
Returns:

  • u (numpy array): Coefficients correspond to each dimension
  • c (double): Constant bias
equadratures.dimension_reduction.standard(X, bounds)¶

Standardize the inputs before Dimension Reduction

Parameters:
  • X – ndarray, the undimensioned input values
  • bounds – ndarray, the upper and lower bounds for input values in each dimension
Returns:

  • X_stnd (numpy array): Standardized input values
equadratures.dimension_reduction.vandermonde(eta, p)¶

Internal function to variable_projection Calculates the Vandermonde matrix using polynomial basis functions

Parameters:
  • eta – ndarray, the affine transformed projected values of inputs in active subspace
  • p – int, the maximum degree of polynomials
Returns:

  • V (numpy array): The resulting Vandermode matrix
  • Polybasis (Poly object): An instance of Poly object containing the polynomial basis derived
equadratures.dimension_reduction.variable_projection(X, f, n, p, gamma=None, beta=None)¶

Variable Projection function to obtain an active subspace in inputs design space

Parameters:
  • X – ndarray, the input
  • f – array, the output
  • n – int, degree of active subspace
  • p – int, degree of polynomials
  • gamma – double, step length reduction factor (0,1)
  • beta – double, Armijo tolerance for backtracking line search (0,1)
Returns:

  • U (ndarray): The active subspace found
