Integrals¶

Utilities for computing integrals of functions

effective_quadratures.integrals.effectivequadratures(stackOfParameters, q_parameter, function)¶

Computes an approximation of the integral using effective-quadratures; this routine uses least squares to estimate the integral.

Parameters:
  • array stackOfParameters (Parameter) – A list of Parameter objects
  • q_parameter (double) – By default, this routine uses a hyperbolic polynomial basis where the q_parameter (value between 0.1 and 1.0) adjusts the number of basis terms to be used for solving the least squares problem.
  • function (callable) – The function whose integral needs to be computed. Can also be input as an array of function values at the quadrature points. The function must be provided either as a callable or an array of values for this routine to work.
Returns:

integral_esq: The effective quadratures approximation of the integral
Return type:

double
Returns:

points: The quadrature points
Return type:

numpy ndarray
effective_quadratures.integrals.sparsegrid(stackOfParameters, level, growth_rule, function=None)¶

Computes a sparse grid of quadrature points based on the distributions for each Parameter in stackOfParameters

Parameters:
  • array stackOfParameters (Parameter) – A list of Parameter objects
  • level (integer) – Level parameter of the sparse grid integration rule
  • growth_rule (string) – Growth rule for the sparse grid. Choose from ‘linear’ or ‘exponential’.
  • function (callable) – The function whose integral needs to be computed. Can also be input as an array of function values at the quadrature points. If the function is given as a callable, then this routine outputs the integral of the function and an array of the points at which the function was evaluated at to estimate the integral. These are the quadrature points. In case the function is not given as a callable (or an array, for that matter), then this function outputs the quadrature points and weights.
Returns:

sparse_int: The sparse grid approximation of the integral
Return type:

double
Returns:

points: The quadrature points
Return type:

numpy ndarray
Returns:

weights: The quadrature weights
Return type:

numpy ndarray
