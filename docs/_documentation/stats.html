Statistics utilities¶

Computing Statistics from Polynomial Expansions

class equadratures.stats.Statistics(coefficients, basis, quadrature_weights=None, polynomial_evals=None, max_sobol_order=None)[source]¶
Parameters:
  • coefficients (numpy-matrix) – Coefficients from a polynomial expansion. Can be computed using any technique.
  • basis (Basis) – Polynomial index set. If an index set is not given, the constructor uses a tensor grid basis of polynomials. For total order and hyperbolic index sets, the user needs to explicity input an index set.
  • quadrature_weights (ndarray) – Length N array. Contains the quadrature weights.
  • polynomial_evals (ndarray) – N by d array, where N is the number of quadrature points; d is the number of dimensions (input parameters). Contains the polynomial basis functions evaluated at the corresponding quadrature points.
  • max_sobol_order (int) – Indicates the maximum order of Sobol’ indices to pre-calculate. In high dimensional problems (e.g. >7D) it is advised to use this option to reduce the computational load upon initialization.
Attributes:
  • self.mean: (double) Mean of the polynomial expansion.
  • self.variance: (double) Variance of the polynomial expansion.
  • self.sobol:(dict) Sobol’ indices of order up to number of dimensions.
